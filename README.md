# Game README

## How to Run

To run the game, use the following command:

```
go run .

```

## Controls

### Player 1

- `W`: Move up
- `A`: Move left
- `S`: Move down
- `D`: Move right
- `H`: Move up-left
- `J`: Move down-left
- `K`: Move down-right
- `L`: Move up-right
- `Space`: Shoot bullet

### Player 2

- `Up Arrow`: Move up
- `Down Arrow`: Move down
- `Left Arrow`: Move left
- `Right Arrow`: Move right
- `Enter`: Shoot

## Multiplayer Mode

To enable multiplayer mode, run the game with the following command:

```
go run . --multi

```
